# reservation-bot

Slack and Alexa bots with Node.js

The following application creates a reservation at "Golden Spoon" restaurant using Slack @resi bot and Amazom Alexa.

## Built With

<ul>
    <li>JavaScript</li>
    <li>Node.js</li>
    <li>Wit.ai</li>
</ul>
